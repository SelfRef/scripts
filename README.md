# Small terminal scripts and tools

## List

Command | Example | Description
--- | --- | ---
bak | bak <file_path> | Adds or removes `.bak` suffix to file name
hf-ls | hf-ls [repo_pattern] | List downloaded HuggingFace models in a tree view
hf-rm | hf-rm <model_id> <filename> | Remove a specific file from a downloaded HuggingFace model
update-media-date | update-media-date <YYYY-MM-DD> | Updates MP4s and JPEGs date in both EXIF and file
